# Cloudflare Tunnel Creation Guide (VM)

Use this guide only to collect the values required by `cloudflare_install_and_setup.sh`.

## 1) Cloudflare Account ID
- Cloudflare Dashboard → **Zero Trust** → **Settings** → **General** → **Account ID**
- Or in the main Cloudflare dashboard sidebar (right side)

## 2) Cloudflare Zone ID
- Cloudflare Dashboard → select your **domain** → **Overview**
- Right sidebar → **Zone ID**

## 3) Cloudflare API Token
Create a **User API Token**:
1) Cloudflare Dashboard → **My Profile** → **API Tokens** → **Create Token**
2) **Create Custom Token**
3) Permissions:
   - Account: **Cloudflare Tunnel → Edit**
   - Zone: **DNS → Edit**
4) Scope it to your **account** and the **zone** for your domain
5) Create token and copy it (shown once)

## Run the script
```bash
sudo -E "/home/malik/self-hosted-server-apps/VM - CICD App Deployment/Cloudflare/cloudflare_install_and_setup.sh"
```
The script will create/reuse/delete the tunnel and create/update the DNS CNAME automatically.
